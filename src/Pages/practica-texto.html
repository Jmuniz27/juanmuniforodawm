Guía básica de etiquetas de texto

Breve repaso de etiquetas HTML que afectan el contenido textual. Ejemplos visibles y descripciones cortas.

Encabezados

h1 — Título principal -> h1

h2 — Subtítulo -> h2

h3 — Sección -> h3

h4 — Subsección -> h4

h5 — Encabezado menor -> h5
h6 — Encabezado más pequeño -> h6

Párrafos y saltos -> h3

Este es un párrafo. Los párrafos se separan automáticamente. -> p

Salto de línea forzado
con la etiqueta br. -> p

Enfasis y semántica inline -> h3

strong -> strong — importancia fuerte.

em -> em — énfasis (cursiva).

mark -> mark — texto resaltado.

small -> small — texto más pequeño.

Citas y referencias -> h3

Esta es una cita en bloque. -> blockquote

Una cita corta: cita entre comillas -> q — atribuida en Fuente Ejemplo -> cite.

Correcciones y cambios -> h3

texto eliminado -> del y texto insertado -> ins.

Subíndice y superíndice -> h3

H2O: H2 -> subO — E=mc2 -> sup

Código y entradas del usuario -> h3

Texto en línea de código: console.log('hola') -> code

Entrada de usuario: Ctrl -> kbd + C -> kbd

Salida de programa: Archivo guardado correctamente -> samp

Variable: n -> var

    // preformateado conserva espacios y saltos
    function ejemplo() {
        return "pre y code";
    }
    -> pre

Abreviaturas y fechas -> h3

HTML -> abbr — uso común de abreviaturas.

Fecha/Time: 20 de octubre de 2025 -> time

Dirección -> h3

Juan Pérez
Ejemplo 123, Ciudad
juan@ejemplo.com -> a
-> address

Listas -> h3

Lista no ordenada -> ul:

  • Elemento uno -> li
  • Elemento dos -> li

Lista ordenada -> ol:

  1. Paso 1 -> li
  2. Paso 2 -> li

Otros elementos útiles -> h3

hr — separador temático -> hr

Ejemplo de details/summary -> summary

Contenido oculto que se muestra al desplegar. -> details
