¿Qué es CSS?

CSS (Cascading Style Sheets) define el estilo visual de HTML: colores, tipografías, tamaño, layout y más.

Selectores comunes y uso

Selector de elemento

p { color: #222; }

p { color: #222; }

Clase

.resaltado { background: #fffbcc; }

.resaltado { background: #fffbcc; padding: 6px; }

ID

#ejemplo-id { color: blue; }

#ejemplo-id { color: blue; }

Atributo

Input de texto:

input[type="text"] { padding: 6px; }

Descendiente

nav a { ... } aplica a enlaces dentro de nav.

nav a { color: inherit; }

Box model y layout

Caja: width, padding, border, margin
.caja { width:200px; padding:10px; border:1px solid #ddd; margin:8px; }

Flexbox básico

Col 1
Col 2
.fila { display:flex; gap:10px; }
.col { flex:1; }

Responsividad

Usa media queries para adaptar estilos a pantallas pequeñas.

@media (max-width:600px) {
  .fila { flex-direction: column; }
}

Consejos rápidos

  • Usa clases para estilos reutilizables.
  • No abuses de IDs para estilos (son específicos).
  • Organiza CSS por componentes o por secciones.
  • Usa herramientas como devtools para inspeccionar y probar cambios.
